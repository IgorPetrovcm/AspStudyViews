ASP.Net core mvc - представляет в общем виде построение приложений вокруг трех основных компонентов - Model(модели) - отвечает за работу с данными
View(представления) - определяеи визуальную состовляющую
Cintroller(контроллеры) - логика обработки запросов
схема: mvc.png

Razor Pages - представляеи модель, при котором за обработку запроса отвечают специальные сущности - страницы Razor Pages. Каждую отдельную сущность можно ассоциировать с отдельной веб-сьраницей

ASP.Net Core Web API - представляет реализацию паттерна REST, при котором для каждого типа http-запроса  (GET, POST, PUT, DELETE) предназначен отдельный ресурс. Подобные ресурсы определяются в виде методов контроллера Web API.

Blazor - представляет фреймворк, который позволяет создавать интерактивные приложения как на стороне сервера, так и на стороне клиента и позволяет задействовать  на уровне браузера низкоуровневый код WebAssembly
